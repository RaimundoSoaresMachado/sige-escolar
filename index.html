Perguntas Frequentes
Tire suas dúvidas sobre o SIGE

O SIGE funciona para qualquer município?

Sim! O sistema é flexível e se adapta a municípios de qualquer porte, desde pequenas cidades com poucas escolas até redes municipais com centenas de unidades.

Como é feita a migração dos dados atuais?

Nossa equipe auxilia na importação dos dados existentes (alunos, turmas, professores) para o sistema, garantindo uma transição tranquila.

Os professores precisam de treinamento especial?

O sistema foi desenvolvido para ser intuitivo. Oferecemos guia de uso integrado e treinamento online para toda a equipe durante a implantação.

Os dados ficam seguros na nuvem?

Absolutamente. Utilizamos criptografia, backups automáticos e controle de acesso rigoroso, em conformidade com a LGPD.

É possível personalizar para a realidade da nossa rede?

Sim. O sistema permite configurações específicas por escola, como avaliação conceitual para os primeiros anos do fundamental, logotipos e muito mais.
